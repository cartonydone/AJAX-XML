This repository contains two mini projects focused on **XML** and **AJAX**, demonstrating client-server interaction and dynamic data handling.

## Projects Included

### 1. XML Parsing
- XML file handling
- Data extraction using XML parsing techniques

### 2. AJAX Search Suggest & Database Select Menu
- AJAX-based search suggestion feature
- Dynamic database select menu
- PHP backend processing
- XML parsing integration

## Topics Covered

- XML
- AJAX
- XML Parsing
- AJAX Search Suggest

## How to Use

1. Clone the repository:
   ```bash
   git clone <repository-url>
````

2. Navigate to the project folders:

   * `1 - XML Parsing`
   * `2 - AJAX Search Suggest`
   * `AJAX - Database Select Menu`

3. Run the project on a local server (recommended for AJAX & PHP).
